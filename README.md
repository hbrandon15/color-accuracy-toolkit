# Camera Color Science Toolkit

## Goal
Measure the color accuracy on a Sony Alpha 7 IV and an iPhone 13 Pro Max camera. 

## Methodology 

**Shoot Setup**

- **Location:** Indoors under controlled light
- **Light source:** Two Aputure studio lights set to 6500K
- **Camera settings:** Manual exposure, no in-camera corrections, Picture Profile set to neutral/flat, lossless compressed RAW
- **Framing:** Chart fills roughly 60-70% of the frame, shot straight-on (no angle), no shadows falling on the patches
- **Lens:** Your 35mm f/1.8 is fine. Stop down to f/5.6 or f/8 for flatness
